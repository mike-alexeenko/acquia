RESPONSIVESLIDES
----------------
Provides a Slideshow formatter for Image fields, using the JQuery ResponsiveSlides.js plugin.
https://drupal.org/project/responsiveslides


CONFIGURATION
-------------
All in Views style settings.

GOOD TO KNOW
------------
Setting a custom "JS/CSS Selector for custom pager navigation" means the elements 
within will be used as pager items (so there better be enough to match the slide count)!
